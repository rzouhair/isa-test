# Common Road Signs

- **Octagonal red + white** — Stop
- **Triangular yellow** — Warning (yield, curve ahead, slippery when wet)
- **Rectangular green** — Directional / guide
- **Rectangular white** — Regulatory (speed limit, one way)
- **Circular yellow w/ black X** — Railroad crossing ahead
- **Pentagonal yellow** — School zone / school crossing
- **Diamond orange** — Construction / work zone
- **Circle w/ red slash** — Prohibited action

### Colors

| Color  | Meaning              |
|--------|----------------------|
| Red    | Stop / prohibited    |
| Yellow | Warning / caution    |
| Green  | Go / directional     |
| White  | Regulatory           |
| Blue   | Services / tourist   |
| Orange | Construction         |
| Brown  | Recreation / parks   |
| Pink   | Incident / emergency |
