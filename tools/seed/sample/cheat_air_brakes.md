# Air Brakes — Quick Reference

### Key components
- **Compressor** — builds air pressure.
- **Governor** — cuts compressor in/out of service (~100–125 psi).
- **Air tanks** — store compressed air; must be drained daily.
- **Brake pedal + foot valve** — applies service brakes.
- **Slack adjusters** — compensate for brake shoe wear.
- **Spring brakes** — apply if air pressure drops below ~20–45 psi.

### Checks
1. Build pressure to governor cut-out.
2. Drain tanks, check for contamination.
3. Low-pressure warning must activate by 60 psi.
4. Spring brakes must apply between 20–45 psi.
5. Static leak rate: ≤2 psi/min (single), ≤3 psi/min (combo) with brakes released.
6. Applied leak rate: ≤3 psi/min (single), ≤4 psi/min (combo).
